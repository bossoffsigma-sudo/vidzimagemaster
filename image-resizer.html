How to Use This Tool

  1. Upload your image using the drag & drop area or file browser
  2. Set your desired width and height in pixels
  3. Adjust quality if needed (higher quality = larger file size)
  4. Toggle aspect ratio lock to maintain proportions
  5. Click "Resize Image" to process
  6. Download your resized image
